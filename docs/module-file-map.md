## Notes

- `COCOM01Y.cpy` is used by all online modules as it carries the CICS COMMAREA session context
- Files under `app/app-authorization-ims-db2-mq/` belong to the optional **authorization** module
- Files under `app/app-transaction-type-db2/` belong to the optional **transaction-type-db2** module
- Files under `app/app-vsam-mq/` belong to the optional **mq-integration** module
- Some copybooks are shared across multiple modules; each mapping is listed separately
