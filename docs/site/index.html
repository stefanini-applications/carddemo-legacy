CardDemo Mainframe Application

Credit Card Management System — COBOL/CICS Documentation

CardDemo is a COBOL/CICS mainframe reference application demonstrating a full-featured credit card management system. It covers online transaction processing via CICS, batch posting and reporting via JCL, and optional extensions for IMS/DB2/MQ integration.
